Nieuw SkyLight theme: Simple

11 September 2021

Ik heb gister een nieuw default theme voor SkyLight gemaakt. SkyLight is mijn website bouwer. Het default theme was tot nu toe “Ugly Blue”. Ugly Blue was een debugtheme om snel dingen te testen. Sinds de basis voor de editor nu af is vond ik het tijd de site die de bezoekers te zien krijgen een opfrissertje te geven.

Het nieuwe theme is gebaseerd op een Blogger theme dat ik voor mijn tweede blog (Stupid Codes) heb gemaakt. Het is een simpel maar mooi theme.

Het nieuwe theme zal default zijn in de volgende SkyLight release. Je kan het ook alvast downloaden via de theme picker: https://robinboers.github.io/SkyLight-themelibrary

 - Robin
